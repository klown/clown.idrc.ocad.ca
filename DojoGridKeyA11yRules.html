Dojo Grid Keyboard A11y Rules

  • Test file:  .../dojox/grid/tests/test_edit.html
  • TAB to the grid.
    • First cell highlights.
    • ARROW keys to navigate around the cells.
    • TAB/SHIFT-TAB also provide navigation among cells.
    • Problem:  how to quickly TAB away from the grid?

ARROW Navigation Detail

  • If cell contains e.g., checkbox, that checkbox appear to have focus.
    • SPACE to change sense of checkbox -- automatically change to edit-mode, and switch checked state.
  • If cell contains editable text, the cell itself retains focus.
    • RETURN to switch to edit-mode.
      • sometimes editable text becomes a combobox.
        • keystrokes apropos a combobox.
      • sometimes editable text becomes a text field.
        • keystrokes apropos a text field.
  • If in edit-mode, cannot ARROW away from cell:
    • RETURN indicates acceptance of change to widget, exits edit-mode, and can then ARROW around.
    • ESC also exits edit-mode, leaves { checkbox in latest state | combobox reverts to original value }, and can then ARROW around.
      • Note:  not sure if ESC is meant to revert widget to original state or not -- inconsistent behaviour.

TAB/SHIFT-TAB Detail

  • Only left/right navigation (ARROWing allows moving up and down rows as well).
  • At end of row and moving forward, moves to first cell in next row.
  • At end of row and moving backward, moves to last cell in previous row.
  • If not in edit-mode, navigating to the next cell simply highlights the next cell.
  • If in edit-mode, navigate to the next cell, highlights that cell and puts it into edit-mode.
    • RETURN indicates acceptance of change to widget, exits edit-mode, and can then ARROW around.
    • ESC also exits edit-mode, leaves { checkbox in latest state | combobox reverts to original value }, and can then ARROW around.

Edit Mode Detail

  • If in edit-mode, keystrokes are apropos for the widget that has focus within the cell:
    • e.g., if a checkbox, SPACE toggles.
    • e.g., if a combobox, ARROWS select different values.
    • e.g., if a text field, ARROW among the characters of the text, or select all of the text and DELETE it, or add/remove new characters to that text by typing.
