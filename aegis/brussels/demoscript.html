Starting GNOME Shell

  1. Start a “Terminal”:
    • Locate the “Screen” icon on the horizontal bar going across the top of the screen (the so-called gnome panel).
    • Click it once.
    • At the prompt, type: “gShell <ret>”
    • Move the mouse to the desktop and wait until it changes from the “busy cursor” state back to an arrow. This may take a while; don't panic.
    • If you want to, you can minimize this window, but do not close it.

Note: to exit GNOME Shell, put focus on this terminal window and press <cntl+c>. This should take you back to the standard desktop environment.

Launch Magnifier Preferences Dialog

Note: You should not launch this dialog before launching GNOME Shell because the “Docked” screen position is handled using D-Bus (service), not Gsettings (preferences). The magnifier D-Bus service is initialized during the launch of the GNOME Shell. It must be running so that the preference dialog can discover the service.

  1. Locate the “Magnifier Preferences” short cut on the right of the desktop.
  2. Double-click it. It should come up as a Window titled “Magnifier Preferences”.
  3. Demonstrate turning Magnfier on and off with “Zoom” radio buttons.
    • Should come up full screen, proportional mouse tracking, lens mode off, and no crosshairs.
  4. Demonstrate the various Mouse Tracking modes.
    • End with “Proportional”.
  5. Demonstrate the various Screen positions.
    • Avoid “Docked” for now, end with “Full Screen”.
  6. Demonstrate magnification factor.
  7. Switch to “Docked”.
  8. Turn on lens mode.
    • Show interaction with different mouse tracking modes.
    • Centred and Proportional immediately move the lens. Push only moves the lens when at edge of magnified view.
  9. Switch to one of the half-screen positions.
    • This is a way to show how larger lenses.
  10. Turn lens mode off.
  11. Switch to full screen.
  12. Demonstrate various crosshairs settings.
    • The reason for crosshairs is to make it easier to locate the mouse cursor.
    • The reason for crosshairs clip is to make it easier to (1) see the mouse cursor, and (2) see text that is immediately below the mouse.
  13. Turn magnifier off using Zoom Off radio button.
  14. Prep for running Orca -- end with these preferences:
    • Proportional mouse tracking,
    • Full Screen,
    • Magnification factor of 2.0,
    • Moveable lens disabled,
    • Scrolling at screen edges disabled,
    • Crosshairs enabled.
    • Crosshairs clip enabled.
    • Blue crosshairs (#0000FF)
    • Any opacity (but 0.5 is nice).
    • Any length.
    • Thickness 16 pixels.

    Launch Orca

    1. Go to “Activities” button in top left of screen.
    2. Type “Orca” return.
      • Orca should start up.
      • Be patient, as it takes a while.
      • Orca has a very simple main window with three buttons: “Preferences”, “Quit”, “About”, and “Help”. Focus should be on the “Preferences” button.
    3. Launch Orca Preferences.
      • Click the “Preferences” button, or press space (assuming Preferences has focus).
      • Orca's (rather large) preferences dialog should appear, but, again, it takes a while.
    4. Focus should be on the “General” Tab.
      • Either arrow right or click the “Magnifier” Tab.
    5. Demonstrate these settings:
      • Left side of panel:
        • Enable magnifier checkbox.
        • Scale factor spinner.
        • Position popup
      • Right side of panel, vertically towards the centre:
        • Enable cross-hair checkbox.
        • Enable cross-hair clip checkbox.
        • Cross-hair size spinner (size means thickness here, not length).
        • Cross-hair colour.
      • Note that all of these are modified using D-Bus. They change the magnifier behaviour, but do not affect the preferences. You will not see changes reflected in “my” Magnifier Preferences dialog. In particular, if you disable the crosshairs using Orca and either close its preferences dialog or quit Orca entirely, then you have to change crosshairs thickness in the Magnifier Preferences dialog to make them visible again.
    1. Exit Orca's preferences dialog.
      • Click “OK” in the bottom right area.
      • This button may not be on screen since the window is very wide, and the Touchpad has a small screen. Move the window to the left.
      • Orca (very) briefly turns off the magnifier. And, It takes a verrry long time for the dialog to close.

    Open Office + Orca + Magnifier

    This is a really good demonstration of focus tracking. You should do this using just the keyboard.

    1. Make sure the crosshairs are enabled with crosshairs clipped also enabled.
    2. Find the icon named “Text Test File.odt” on the bottom left of the desktop.
    3. Double click it; OpenOffice should launch.
    4. Focus should come up in the (empty) document at the text input cursor, and it should be automatically placed within view.
    5. Start typing. If you type enough, then the magnified view should shift leftwards to keep the text caret at or near the centre of the view.
    1. Hit <alt> to go to the menu bar. The view should shift to the “File” menu.
    2. Navigate the menu bar using left/right arrow keys. The view shifts such that the highlighted menu is at or near the centre of the magnified view.
    3. Open the “File” menu by navigating to it and then using the down arrow key.
    4. Navigate the “File” menu. The view shifts such that the highlighted menu item is at or near the centre of view.
    5. (Optional) Press <esc> twice to first close the menu, and then to leave the menu bar. The focus returns to the input caret.
    6. Use the keyboard to navigate to the “File – Exit” menu item. Press return to quit OpenOffice.
    7. A “Do you want to save your changes” dialog appears, with keyboard focus on the “Save” button.
    8. Tab to the “Discard” button and press space. This should quit without saving.

    One final note: I almost never demonstrate the "Scroll at screen edges" checkbox in the Magnifier Preferences dialog. It only works with centred mouse tracking. I just discovered a new user for it, however: Due to the small size of the Touchpad screen, it is sort of useful to see when you are at or beyond the edge of the screen.
