Component Name: Slider

States: Up, Down

 

Description

This is used to allow users to set the value of some multivalued dimension; for example, a volume control could be represented as a slider.

Below is the Swing Set slider sample showing both horizontal and vertical sliders which might appear as options.

 

 

Sound Rationale

There is a "generic" slider sound which is like one of the old sliding whistles kids used to play with. Once in a position where a slider can be "activated" either up or down, a chromatic scale takes over and tones are sounded either up or down depending on the direction the computer user is going through the slider options. By using this scale, the computer user can receive feedback about the "amount" of sliding that is going on, either in small or large increments.

The slider up or down sounds are made by combining the following chormatic tones to indicate an up or down pattern. The tones are listed below in numerical order, while in the forms they are referred to as "slider up sound" or "slider down sound" as their range will depend on the values moved up or down.
