Component Name: ToggleButton
States: On and Off

Description

An widget that represents a button that remains pushed-in when pushed, or pops out when pushed a second time. An example would be a tool bar button in a word processor that "toggles" between bold text being turned "on" or "off" affecting selected text. The Swing Set examples are a titch more creative and actually animate elements. Again, there are Text ToggleButtons and Image ToggleButtons.
