Menu Button

A menu button is a button that posts a menu when activated. The button is not attached to a menu bar. The first picture below shows a menu button circled in red. Note that the menu button forms part of the label for the checkbox, namely, "Annouce the time: On the half hour".

Screenshot of a menu button

The next picture shows the menu invoked when the button is activated. The text alternative of the embedded button is the text it has when the menu is not posted; hence, in the first picture, the text alternative of the menu button is "On the half hour".

Screenshot of a menu button with associated menu posted

Note that there is a menu button design pattern in the WAI-ARIA Authoring Practices Guide.
