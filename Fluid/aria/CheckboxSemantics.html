Checkboxes: What is Published to A11y API?

<input type="checkbox" role="radio" aria-checked="false" checked>:

MSAA: radio button, checked. Default action 'uncheck'

<input type="checkbox" aria-checked="mixed">:

MSAA: checkbox, no checked state reported. Default action is 'check'

<input type="checkbox" aria-checked="true">:

MSAA: checkbox, no checked state reported. Default action is 'check'

<div role="checkbox" aria-checked="mixed">:
[ ] Aria checkbox with mixed checked state

MSAA: checkbox, no checked state reported. Default action is 'check'
