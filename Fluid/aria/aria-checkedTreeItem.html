Element with @role='treeitem' with aria-checked='mixed'

Directions

  1. Tab into the tree control.
  2. Use the up/down arrow keys to change the active descendant.
  3. Use the right/left arrow keys to expand/collapse appropriate nodes.

Note: this example has been updated to account for ARIA's newly defined accessible name calculation, so at the date of this writing (2008/12/14), it does not work as intended with any web browser.
